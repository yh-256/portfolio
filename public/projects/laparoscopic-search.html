自然言語クエリに合う術中場面を、視覚意味と時間的文脈の両方から探索する。Retrieve intraoperative scenes from natural-language queries using both visual semantics and temporal context.

01 / Context

100時間の動画から、必要な数秒へたどり着く。Navigate from one hundred hours of video to the few seconds that matter.

長時間の手術動画から特定場面を探す作業は、教育・振り返り・研究のいずれでも大きな負担になります。単純なテキスト検索では映像内容を扱えず、視覚類似度だけでは術式の時間的な流れを無視してしまいます。Finding a specific moment in long surgical video is costly for education, review, and research. Text search cannot see the scene, while visual similarity alone ignores procedural chronology.

本研究では、視覚言語埋め込みによる初期検索に、クラスタ構造と時間情報を用いた再検索・再順位付けを組み合わせました。The method combines initial vision-language retrieval with cluster-aware re-search and temporal reranking.

02 / Method

似ている一枚ではなく、場面として妥当な位置を探す。Find the right moment, not merely the most similar frame.

  1. Offline indexing
    5秒間隔で抽出したフレームを埋め込み、近傍検索可能な索引を構築。Embed frames sampled every five seconds and build a nearest-neighbor index.
  2. Initial retrieval
    自然言語クエリとフレーム埋め込みの類似度から候補を取得。Retrieve candidates using query–frame embedding similarity.
  3. Cluster-aware refinement
    有望な候補周辺のクラスタを再探索し、意味的に関連する場面を補完。Re-search promising clusters to recover semantically related moments.
  4. Temporal reranking
    術式の時間的な位置関係を重みとして順位へ反映。Use procedural temporal position as an additional ranking signal.

03 / Result

時間情報は、検索順位を一段押し上げた。Temporal context improved the final ranking.

時間重みを組み込んだ順位スコアは 0.815、Text-onlyベースラインは 0.755 でした。一方で、初期検索や改善量の一部指標ではベースラインが優位なケースもあり、時間重みが常に万能ではないことも確認しました。The temporal-weighted ranking score reached 0.815, compared with 0.755 for the text-only baseline. Some initial-retrieval and improvement metrics still favored the baseline, showing that temporal weighting is not universally beneficial.

結果は、視覚意味だけでなく「手術のどの段階か」を扱う価値を示します。同時に、クエリによって時間情報の有効性が異なるため、今後はクエリ適応的な重み付けが課題です。The result supports modeling not only what a frame shows, but where it occurs in the procedure. Future work is query-adaptive temporal weighting.
